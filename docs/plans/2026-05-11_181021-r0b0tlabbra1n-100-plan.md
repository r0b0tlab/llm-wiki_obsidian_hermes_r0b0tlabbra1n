### Task 1.2: Return real BM25 scores

**Files:**
- Modify: `r0b0tlabbra1n/index/sqlite_index.py`

**Implementation:**
Use FTS5 bm25:

```sql

       snippet(pages_fts, 5, '<mark>', '</mark>', '...', 32) AS snippet,

FROM pages_fts

ORDER BY rank ASC
LIMIT ?
```

Expose `score = -rank` or `rank = rank`; be consistent in docs.
